# Domain Knowledge

## Key Terms
- DIRE = Indonesian REIT structure under OJK
- SPV = Special Purpose Vehicle
- KIK = Collective Investment Contract
- Economic Rights Token = Token representing revenue or debt claim
- Sukuk Ijarah = Lease-based Islamic bond

## Privacy Model
- Personal data stored off-chain
- On-chain only stores cryptographic reference
- Data Controller defined at SPV level

## Liquidity Strategy
- Dual market model
- Regulated exchange listing
- NAV oracle integration
- Redemption mechanism required
