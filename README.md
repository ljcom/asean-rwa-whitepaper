# ASEAN RWA Whitepaper Workspace

This repository contains two main components:

- `paper/`: Source materials and build assets for the whitepaper.
- `renextnft/`: Supporting product stack (frontend, backend, smart contracts).

## Quick Start

- For the whitepaper workflow, see `paper/README.md` https://github.com/ljcom/asean-rwa-whitepaper/blob/main/paper/README.md.
- For the whitepaper PDF, see https://github.com/ljcom/asean-rwa-whitepaper/blob/main/paper/04-build/whitepaper.pdf.
- For the app stack, see `renextnft/development.md` or `renextnft/run.sh`.
