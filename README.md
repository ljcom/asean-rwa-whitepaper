# ASEAN RWA Whitepaper (Real Estate Tokenization)

Working draft for: **A Regulatory-Aligned Framework for Cross-Border Real Estate Economic Rights Tokenization in ASEAN**.

## Scope (Current Draft)

- Minimum scope: ASEAN (phased rollout), **Indonesia-first pilot**
- Underlying asset jurisdiction: **Indonesia**
- Token represents **economic rights only** (no land title transfer via blockchain)
- Hybrid on-chain/off-chain compliance architecture (KYC off-chain, restricted transfers, auditability)
- Funding models: debt, profit participation (REIT-like), sukuk (later phase), crowdfunding
- Liquidity engineering emphasis (not automatic liquidity); secondary market is venue-specific and not an assurance of exit

## Repository Structure

- `00-meta/` — outline, glossary, assumptions, changelog, decision log
- `01-draft/` — main whitepaper sections (including `14-technical-annex.md`)
- `02-figures/` — Mermaid diagrams and tables (Markdown sources)
- `03-references/` — reference placeholders (`sources.md`, `refs.bib`)
- `04-build/` — build outputs and build script

## Build (Basic)

Prerequisite: `pandoc` installed (available in this environment).

Run:

```bash
./04-build/build.sh
```

Outputs:

- `04-build/whitepaper.generated.md`
- `04-build/whitepaper.docx`
- `04-build/whitepaper.html`

Notes:

- PDF is not generated by default unless a PDF engine (e.g., `xelatex`) is installed.
- Mermaid diagrams are included as code blocks unless rendered to images as part of the build pipeline.

## Mermaid Diagrams (Optional Rendering)

If `mmdc` is installed, the build script will attempt to render Mermaid code blocks into PNG images under `04-build/diagrams/` and embed them into `04-build/whitepaper.rendered.md`.

If rendering fails in your environment, you can still build outputs (diagrams remain as code blocks), or rerun rendering manually:

```bash
python3 04-build/render_mermaid.py 04-build/whitepaper.generated.md 04-build/whitepaper.rendered.md 04-build/diagrams
```

Then compile from the rendered Markdown:

```bash
pandoc 04-build/whitepaper.rendered.md --toc --toc-depth=3 -o 04-build/whitepaper.docx
pandoc 04-build/whitepaper.rendered.md --toc --toc-depth=3 --pdf-engine=/Library/TeX/texbin/xelatex -o 04-build/whitepaper.pdf
```
